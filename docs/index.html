Multi-Agent Prototype Builder

This website documents the production-style multi-agent pipeline in this repository. The system converts a product idea into a planned, implemented, tested, and evaluated prototype.

How it works

  1. Orchestrator sets milestones and coordinates retries/failures.
  2. Specification agent generates PRD, features, stories, and contracts.
  3. Architecture agent defines boundaries and tradeoffs.
  4. Implementation agent writes runnable prototype files.
  5. Testing agent runs unit/integration suites and reports failures.
  6. Evaluation agent reports quality/risk/debt and next steps.

Run locally

PYTHONPATH=src python -m mas.cli "Your product idea" --output build_outputs

pytest -q

See the repository README for full usage and scaling guidance.
